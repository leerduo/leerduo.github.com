Create Lists

The RecyclerView widget is a more advanced and flexible version of ListView. This widget is a container for displaying large data sets that can be scrolled very efficiently by maintaining a limited number of views. Use the RecyclerView widget when you have data collections whose elements change at runtime based on user action or network events.

RecyclerView是ListView的更高级更灵活的版本。该控件通过保持(maintaining)有限(limited)数量的view来保证在展示大量数据的时候可以保持滑动流畅。RecyclerView提供一下两个简化和处理数据：

  • LayoutManager去定位Items
  • 默认的item操作动画,比如移除或者添加item

同时我们可以自己去定义LayoutManager和动画。

使用RecyclerView控件,必须指定adapter和LayoutManager。创建adapter的时候，继承自RecyclerView.Adapter。具体的实现依赖于要显示的view。

LayoutManager在RecyclerView中定位了item的具体位置,决定了什么时候去复用view，什么时候item的view对用户而言不可见。复用view的时候，LayoutManager会去询问(ask)adapter从数据源中去替换view的内容。RecyclerView这样从两个角度进行的优化。一是避免了创建不需要的view，二是避免了执行耗时的findViewById()查找。

RecyclerView 提供了这些内置的LayoutManager

  • LinearLayoutManager
  • GridLayoutManager
  • StaggeredGridLayoutManager (瀑布流)

创建自定义的LayoutManager,需要继承RecyclerView.LayoutManager。
